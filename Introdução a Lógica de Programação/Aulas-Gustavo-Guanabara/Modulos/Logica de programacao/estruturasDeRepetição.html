Estruturas de Repetição

"O Sorteador de Números é uma ferramenta online projetada para facilitar a realização de sorteios. Com uma interface intuitiva e amigável, os usuários podem definir o intervalo de números desejado e, em seguida, clicar no botão 'Sortear' para obter números aleatórios instantaneamente. É perfeito para sorteios de brindes, rifas ou qualquer situação em que números precisem ser sorteados de forma justa e imparcial."
